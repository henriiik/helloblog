## Rocket

* `cargo.toml`

  ```toml
  [dependencies]
  rocket = "0.3.6"
  rocket_codegen = "0.3.6"
  ```

* `src/main.rs`

  ```rust
  #![feature(plugin)]
  #![plugin(rocket_codegen)]

  extern crate rocket;

  #[get("/")]
  fn index() -> &'static str {
      "Hello, world!"
  }

  fn main() {
      rocket::ignite().mount("/", routes![index]).launch();
  }
  ```

* it works!
